# IAforbet

## Description

Ce code illustre comment scraper les divers championnats à utiliser dans les modèles de IAforbet.

## Installation

1. Clonez le dépôt :

   ```bash
   git clone https://github.com/Arnel7/bd-foot.git
   ```

2. Installez les dépendances :
   ```bash
   pip install -r requirements.tx
   ```
